Random song

To listen to a random song, click here, or use the normal url of the webpage wihtout any parameters.

Specific song

To listen to a song from a specific model, use the url of the webpage with parameters /song/{model_name}/{instrument}.

To generate a song

To generate a song from a model, use the url of the webpage with parameters /generate/{model_name}/{instrument}.
Number of songs is an optional parameter: /generate/{model_name}/{instrument}/{count} will generate {count} songs.
Temperature is an another optional parameter: /generate/{model_name}/{instrument}/{count}/{temp} will generate songs with {temp} sampling temperature. It is a floating point number, which is 0.8 by default. The higher it is, the more unpredictable the output will be, the lower it is, the more repetitive the output will be.
